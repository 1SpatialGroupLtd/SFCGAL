About

SFCGAL is a C++ wrapper library around CGAL with the aim of supporting ISO 191007:2013 and OGC Simple Features for SQL 1.2 for 3D operations.

SFCGAL provides standard compliant geometry types and operations, that can be accessed from PostGIS or through its C and C++ APIs.

Among supported geometry types are (in both 2D and 3D) :

  • Points
  • LineStrings
  • Polygons
  • TriangulatedSurfaces
  • PolyhedralSurfaces
  • GeometryCollections
  • Solids

Supported operations include :

  • WKT reading and writing (including exact rational number representation for coordinates),
  • intersection operations and predicates,
  • convex hull computation,
  • tessellation,
  • extrusion,
  • area and distance computation,
  • Minkovski sums,
  • offsets and straight skeleton generations.
